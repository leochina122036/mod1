| 项目主图 | 800×500px | <500KB |
| 项目详图 | 600×400px | <300KB |
| 证书 | 800×600px | <400KB |

## 🚀 使用步骤

1. 阅读 `图片需求说明.md` 了解每张图片的要求
2. 准备图片并转换为 `.webp` 格式
3. 按照规定名称重命名文件
4. 将所有图片放入本文件夹
5. 刷新浏览器查看效果
