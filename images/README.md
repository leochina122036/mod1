# 图片文件夹使用说明

## 📋 页面架构与栏目设计理念

### 为什么需要这些栏目？

1. **Banner区域** - 第一印象决定成败
   - 目的：3秒内抓住HR/招聘者注意力
   - 展示：核心价值主张 + 关键数据 + 行动召唤
   - 图片：avatar.webp（建立信任感）

2. **基本信息** - 快速筛选候选人
   - 目的：让HR在30秒内了解基本情况
   - 展示：年龄、性别、驾照、联系方式等关键信息

3. **教育背景** - 证明学习能力
   - 目的：展示持续学习能力和与时俱进的态度
   - 从2024年AI学习到2000年本科，15年学习历程
   - 突出：AI、安全、SEO、开发的全栈学习路径

4. **技能特长** - 核心竞争力展示
   - 目的：让HR清楚知道候选人"能做什么"
   - 采用进度条可视化：精通95%、熟练80%
   - 每项技能配具体成就描述，用数据说话

5. **工作经历** - 职业发展轨迹
   - 目的：展示稳定的职业上升路径
   - 时间轴设计：视觉化展示15年职业发展
   - STAR法则：职责 + 成就 + 证明人

6. **项目经验** - 实战能力证明
   - 目的：用实际案例证明解决问题能力
   - STAR法则：背景 → 方案 → 成果（数据化）
   - 图片：ai-app1-4.webp（增强可信度）

7. **AI应用** - 展示前沿技术能力
   - 目的：突出AI方向专长，符合市场热点
   - 4个完整应用案例 + 图片展示
   - 图片：ai-app1-4.webp（视觉化能力）

8. **证书展示** - 增加可信度
   - 目的：第三方机构背书，增强信任
   - 图片：cert1-4.webp（权威认证）

9. **个人照片** - 增加亲和力
   - 目的：让简历更有温度，建立情感连接
   - 图片：photo1-2.webp（工作照+生活照）

10. **联系方式** - 行动召唤
    - 目的：降低联系门槛，促进转化
    - 多种联系方式 + 快捷操作按钮
